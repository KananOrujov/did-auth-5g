ATTACK 01 STRIDE-R
Replay Attack
Submits 5 rapid requests for the same valid SUPI simultaneously, testing nonce-per-proof enforcement.
Expected: All requests handled correctly. Nonce binding prevents proof reuse.
ATTACK 02 STRIDE-S
Impersonation
Sends authentication requests for SUPIs with no credential in the holder wallet (imsi-...999).
Expected: Denied immediately — no_credential_mapped, no proof exchange.
ATTACK 03 STRIDE-E
Revoked Credential
Attempts authentication using UE-001 whose credential was revoked on the ledger.
Expected: Denied in ~75ms via pre-check — proof exchange skipped.
ATTACK 04 STRIDE-E
Slice Policy Bypass
UE-006 holds a valid SST:2 credential and attempts to access the SST:1 network slice.
Expected: Denied — slice_policy_denied. Attribute is cryptographically bound.
ATTACK 05 STRIDE-D
Denial of Service / Load Attack
Fires 8 concurrent mixed requests simultaneously — valid UEs, unknown SUPIs, revoked UE, wrong-slice UE. Tests queue limit, semaphore, and fail-close under load.
Expected: All requests handled correctly with no crashes. Valid UEs allowed, denied UEs rejected with correct reasons.
